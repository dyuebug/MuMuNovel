# 01 - 项目概览

## 欢迎来到 MuMuNovel

MuMuNovel 是一个基于 AI 的智能小说创作助手，旨在帮助作者通过 AI 技术提升创作效率。

## 项目定位

这是一个**全栈 Web 应用**，采用前后端分离架构：
- **后端**：Python FastAPI，提供 RESTful API 和 SSE 流式接口
- **前端**：React + TypeScript，提供现代化的用户界面
- **数据库**：PostgreSQL（生产）/ SQLite（开发）
- **AI 集成**：支持 OpenAI、Gemini、Claude 等多种 AI 模型

## 核心功能

### 1. 智能创作向导
- 自动生成小说大纲
- AI 辅助角色设定
- 世界观构建

### 2. 角色管理系统
- 角色信息管理
- 角色关系可视化
- 组织架构图

### 3. 章节编辑器
- 章节创建与编辑
- AI 辅助续写
- 内容润色优化
- 重新生成功能

### 4. 伏笔管理
- 伏笔追踪
- 时间线管理
- 情节关联

### 5. 提示词工坊
- 社区驱动的 Prompt 分享平台
- 自定义提示词模板
- 提示词评分与收藏

### 6. 职业等级体系
- 修仙境界设定
- 魔法等级系统
- 自定义等级体系

## 适合谁学习？

### 初学者
如果你是编程初学者，建议先学习：
1. **Python 基础**：变量、函数、类、异步编程
2. **JavaScript/TypeScript 基础**：ES6+ 语法、Promise、async/await
3. **Web 基础**：HTTP 协议、RESTful API、前后端交互

### 进阶开发者
如果你已有一定基础，可以直接学习：
1. **FastAPI 框架**：路由、依赖注入、异步处理
2. **React Hooks**：useState、useEffect、自定义 Hooks
3. **数据库设计**：SQLAlchemy ORM、数据库迁移
4. **AI 集成**：API 调用、流式响应、Prompt 工程

## 学习路径建议

### 路径 1：前端开发者
1. 从前端代码开始（`frontend/src/`）
2. 理解 React 组件结构
3. 学习 API 调用方式（`services/api.ts`）
4. 逐步了解后端 API 设计

### 路径 2：后端开发者
1. 从后端代码开始（`backend/app/`）
2. 理解 FastAPI 路由和依赖注入
3. 学习数据库模型设计
4. 了解 AI 服务集成

### 路径 3：全栈开发者
1. 先理解整体架构（阅读 CLAUDE.md）
2. 搭建开发环境
3. 从一个完整功能模块入手（如角色管理）
4. 前后端同步学习
