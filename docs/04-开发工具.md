#### VS Code 配置

创建 `.vscode/settings.json`：

```json
{
  // Python 配置
  "python.defaultInterpreterPath": "${workspaceFolder}/backend/.venv/bin/python",
  "python.linting.enabled": true,
  "python.linting.pylintEnabled": false,
  "python.linting.flake8Enabled": true,
  "python.formatting.provider": "black",
  "python.formatting.blackArgs": ["--line-length", "100"],

  // TypeScript 配置
  "typescript.tsdk": "node_modules/typescript/lib",
  "typescript.enablePromptUseWorkspaceTsdk": true,

  // 编辑器配置
  "editor.formatOnSave": true,
  "editor.codeActionsOnSave": {
    "source.fixAll.eslint": true
  },
  "editor.tabSize": 2,
  "editor.rulers": [80, 100],

  // 文件配置
  "files.exclude": {
    "**/__pycache__": true,
    "**/*.pyc": true,
    "**/.pytest_cache": true,
    "**/node_modules": true,
    "**/.venv": true
  },

  // Git 配置
  "git.autofetch": true,
  "git.confirmSync": false,

  // 其他
  "terminal.integrated.defaultProfile.windows": "PowerShell",
  "terminal.integrated.defaultProfile.linux": "bash"
}
```

## API 测试工具

### 1. Postman

**下载地址**：https://www.postman.com/downloads/

**使用方法**：

1. 创建新的 Collection：`MuMuAINovel API`
2. 添加环境变量：
   ```
   base_url: http://localhost:8000
   token: your_jwt_token
   ```
3. 创建请求示例：

**获取项目列表**：
```
GET {{base_url}}/api/projects
Headers:
  Authorization: Bearer {{token}}
```

**创建项目**：
```
POST {{base_url}}/api/projects
Headers:
  Authorization: Bearer {{token}}
  Content-Type: application/json
Body:
{
  "name": "测试项目",
  "description": "API 测试",
  "genre": "玄幻"
}
```

### 2. Thunder Client（VS Code 插件）

**优势**：无需离开 VS Code

**使用方法**：

1. 安装插件后，点击左侧闪电图标
2. 创建新请求
3. 设置环境变量
4. 保存请求到 Collection

### 3. REST Client（VS Code 插件）

**优势**：使用文本文件管理 API 请求

创建 `api-test.http`：

```http
### 变量定义
@baseUrl = http://localhost:8000
@token = your_jwt_token

### 健康检查
GET {{baseUrl}}/health

### 登录
POST {{baseUrl}}/api/auth/login
Content-Type: application/json

{
  "username": "admin",
  "password": "your_password"
}

### 获取项目列表
GET {{baseUrl}}/api/projects
Authorization: Bearer {{token}}

### 创建项目
POST {{baseUrl}}/api/projects
Authorization: Bearer {{token}}
Content-Type: application/json

{
  "name": "测试项目",
  "description": "通过 REST Client 创建",
  "genre": "科幻"
}

### SSE 流式请求（需要特殊处理）
POST {{baseUrl}}/api/ai/generate-stream
Authorization: Bearer {{token}}
Content-Type: application/json

{
  "prompt": "写一段修仙描述",
  "max_tokens": 200
}
```

### 2. Black（代码格式化）

**安装**：
```bash
pip install black
```

**使用**：
```bash
# 格式化单个文件
black app/main.py

# 格式化整个目录
black app/

# 检查但不修改
black --check app/
```

**配置** `pyproject.toml`：
```toml
[tool.black]
line-length = 100
target-version = ['py311']
include = '\.pyi?$'
```

### 3. Flake8（代码检查）

**安装**：
```bash
pip install flake8
```

**使用**：
```bash
# 检查代码
flake8 app/

# 忽略特定错误
flake8 --ignore=E501,W503 app/
```

**配置** `.flake8`：
```ini
[flake8]
max-line-length = 100
exclude = .git,__pycache__,.venv,alembic
ignore = E203,E266,E501,W503
```

### 4. pytest（测试框架）

**安装**：
```bash
pip install pytest pytest-asyncio pytest-cov
```

**使用**：
```bash
# 运行所有测试
pytest

# 运行特定测试
pytest tests/test_api.py

# 生成覆盖率报告
pytest --cov=app tests/
```

## 工具配置最佳实践

### 1. 统一团队配置

将配置文件提交到仓库：
```
.vscode/
  settings.json
  extensions.json
  launch.json
.editorconfig
.prettierrc
.eslintrc.js
pyproject.toml
.flake8
```

### 2. 使用 EditorConfig

创建 `.editorconfig`：
```ini
root = true

[*]
charset = utf-8
end_of_line = lf
insert_final_newline = true
trim_trailing_whitespace = true

[*.{py}]
indent_style = space
indent_size = 4

[*.{js,jsx,ts,tsx,json,yml,yaml}]
indent_style = space
indent_size = 2

[*.md]
trim_trailing_whitespace = false
```

### 3. Git Hooks

使用 pre-commit 自动检查：

**安装**：
```bash
pip install pre-commit
```

**配置** `.pre-commit-config.yaml`：
```yaml
repos:
  - repo: https://github.com/pre-commit/pre-commit-hooks
    rev: v4.4.0
    hooks:
      - id: trailing-whitespace
      - id: end-of-file-fixer
      - id: check-yaml
      - id: check-added-large-files

  - repo: https://github.com/psf/black
    rev: 23.3.0
    hooks:
      - id: black

  - repo: https://github.com/pycqa/flake8
    rev: 6.0.0
    hooks:
      - id: flake8
```

**启用**：
```bash
pre-commit install
```
