# 03 - 快速开始

本文档将带你快速上手 MuMuAINovel，通过实际操作理解项目的核心功能。

## 前置条件

在开始之前，请确保：
- [x] 已完成环境搭建（参考 02-环境搭建.md）
- [x] 后端服务正常运行（http://localhost:8000）
- [x] 前端服务正常运行（http://localhost:5173）
- [x] 已配置至少一个 AI API Key

## 第一次运行

### 1. 启动服务

**方式一：使用 Docker Compose（推荐）**

```bash
# 在项目根目录
docker-compose up -d

# 查看服务状态
docker-compose ps

# 查看日志
docker-compose logs -f backend
docker-compose logs -f frontend
```

**方式二：手动启动**

```bash
# 终端 1：启动后端
cd backend
source .venv/bin/activate  # Windows: .venv\Scripts\activate
python -m uvicorn app.main:app --host localhost --port 8000 --reload

# 终端 2：启动前端
cd frontend
npm run dev
```

### 2. 访问应用

打开浏览器访问：
- **前端界面**：http://localhost:5173
- **API 文档**：http://localhost:8000/docs
- **健康检查**：http://localhost:8000/health

### 3. 首次登录

**使用本地账户登录**：
1. 打开前端页面
2. 点击"登录"按钮
3. 输入在 `.env` 中配置的用户名和密码
   - 默认用户名：`admin`
   - 密码：你在 `.env` 中设置的 `LOCAL_AUTH_PASSWORD`

**使用 LinuxDO OAuth 登录**（可选）：
1. 需要先在 LinuxDO 注册应用获取 Client ID 和 Secret
2. 在 `.env` 中配置 OAuth 信息
3. 点击"使用 LinuxDO 登录"

## 创建第一个项目

### 1. 新建项目

1. 登录后，点击"新建项目"按钮
2. 填写项目信息：
   ```
   项目名称：我的第一部小说
   项目描述：一个关于冒险的故事
   类型：玄幻
   ```
3. 点击"创建"

### 2. 使用智能向导

**自动生成大纲**：

1. 进入项目详情页
2. 点击"智能向导"标签
3. 选择"生成大纲"
4. 输入创作意图：
   ```
   写一个关于少年修仙的故事，主角从小村庄出发，
   经历各种磨难，最终成为一代宗师。
   ```
5. 点击"生成"，等待 AI 生成大纲
6. 查看生成结果，可以编辑或重新生成

**生成角色设定**：

1. 在智能向导中选择"生成角色"
2. 输入角色描述：
   ```
   主角：林风，16岁，来自青云村，性格坚韧不拔
   ```
3. AI 会自动生成详细的角色设定
4. 点击"保存到角色库"

**生成世界观**：

1. 选择"生成世界观"
2. 输入世界观要素：
   ```
   修仙世界，分为凡人界、修仙界、仙界三个层次
   ```
3. AI 生成详细的世界观设定
4. 保存到项目设定中

## 管理角色

### 1. 创建角色

1. 点击"角色管理"标签
2. 点击"新建角色"
3. 填写角色信息：
   ```
   姓名：林风
   年龄：16
   性别：男
   职业：修仙者
   境界：炼气期
   性格：坚韧、善良、有正义感
   背景：青云村普通少年，因缘际会踏上修仙之路
   ```
4. 点击"保存"

### 2. 设置角色关系

1. 在角色列表中选择两个角色
2. 点击"添加关系"
3. 选择关系类型：
   - 师徒关系
   - 朋友关系
   - 敌对关系
   - 亲属关系
4. 填写关系描述
5. 保存后可在关系图中查看

### 3. 查看角色关系图

1. 点击"关系图"按钮
2. 查看可视化的角色关系网络
3. 可以拖动节点调整布局
4. 点击节点查看角色详情

## 编写章节

### 1. 创建章节

1. 点击"章节管理"标签
2. 点击"新建章节"
3. 填写章节信息：
   ```
   章节标题：第一章 踏上修仙路
   章节序号：1
   ```
4. 点击"创建"

### 2. 使用 AI 辅助写作

**AI 续写**：

1. 在章节编辑器中输入开头：
   ```
   青云村，一个偏僻的小村庄。
   林风站在村口，望着远方的群山，心中充满了向往。
   ```
2. 选中文本，点击"AI 续写"
3. 选择续写风格：
   - 详细描写
   - 快节奏
   - 对话为主
4. 等待 AI 生成续写内容
5. 选择满意的内容插入

**内容润色**：

1. 选中需要润色的段落
2. 点击"润色"按钮
3. 选择润色方向：
   - 增强描写
   - 优化语言
   - 调整节奏
4. 查看润色结果，选择应用或重新生成

**重新生成**：

1. 选中不满意的段落
2. 点击"重新生成"
3. 提供修改建议：
   ```
   增加更多心理描写，突出主角的内心挣扎
   ```
4. AI 根据建议重新生成内容

### 3. 保存和导出

1. 点击"保存"按钮保存章节
2. 点击"导出"可以导出为：
   - TXT 文本
   - Markdown 格式
   - Word 文档（.docx）

## 管理伏笔

### 1. 添加伏笔

1. 在章节编辑器中选中伏笔相关的文本
2. 点击"添加伏笔"
3. 填写伏笔信息：
   ```
   伏笔标题：神秘玉佩
   伏笔内容：林风在山洞中发现一块古老的玉佩
   预计揭示章节：第十章
   ```
4. 保存伏笔

### 2. 追踪伏笔

1. 点击"伏笔管理"标签
2. 查看所有伏笔列表
3. 伏笔状态：
   - 🟡 未揭示
   - 🟢 已揭示
   - 🔴 已废弃
4. 点击伏笔查看详情和相关章节

### 3. 揭示伏笔

1. 在对应章节中编写伏笔揭示内容
2. 在伏笔管理中找到该伏笔
3. 点击"标记为已揭示"
4. 填写揭示章节和揭示内容

## 使用提示词工坊

### 1. 浏览提示词

1. 点击"提示词工坊"标签
2. 浏览社区分享的提示词模板
3. 按分类筛选：
   - 角色生成
   - 情节构思
   - 场景描写
   - 对话生成

### 2. 使用提示词

1. 选择一个提示词模板
2. 点击"使用"
3. 填写模板中的变量：
   ```
   模板：生成一个{类型}角色，性格{性格}，背景{背景}

   填写：
   类型：反派
   性格：阴险狡诈
   背景：名门正派叛徒
   ```
4. 点击"生成"，AI 根据提示词生成内容

### 3. 创建自定义提示词

1. 点击"创建提示词"
2. 填写提示词信息：
   ```
   标题：生成修仙境界描述
   分类：世界观设定
   内容：
   请生成一个修仙境界的详细描述，包括：
   - 境界名称：{境界名称}
   - 修炼难度：{难度}
   - 境界特征：{特征}
   - 突破方法：{方法}
   ```
3. 保存并分享到社区

## 配置职业等级体系

### 1. 创建等级体系

1. 点击"设置" → "职业等级"
2. 点击"新建等级体系"
3. 选择类型：
   - 修仙境界
   - 魔法等级
   - 武者等级
   - 自定义
4. 填写等级信息：
   ```
   体系名称：修仙境界
   等级列表：
   1. 炼气期
   2. 筑基期
   3. 金丹期
   4. 元婴期
   5. 化神期
   ```
5. 保存等级体系

### 2. 应用到角色

1. 在角色管理中编辑角色
2. 选择"职业等级体系"
3. 设置当前等级：炼气期
4. 保存后角色卡片会显示等级信息

## 数据导入导出

### 1. 导出项目

1. 在项目列表中选择项目
2. 点击"导出"按钮
3. 选择导出格式：
   - JSON（完整数据）
   - ZIP（包含所有文件）
4. 下载导出文件

### 2. 导入项目

1. 点击"导入项目"
2. 选择之前导出的文件
3. 确认导入
4. 系统自动恢复项目数据

### 3. 批量导出角色

1. 在角色管理中选择多个角色
2. 点击"批量导出"
3. 选择导出格式（JSON/CSV）
4. 下载角色数据

## 探索 API 文档

### 1. 访问 Swagger UI

打开浏览器访问：http://localhost:8000/docs

### 2. 测试 API

**获取项目列表**：

1. 找到 `GET /api/projects` 接口
2. 点击"Try it out"
3. 点击"Execute"
4. 查看响应结果

**创建项目**：

1. 找到 `POST /api/projects` 接口
2. 点击"Try it out"
3. 填写请求体：
   ```json
   {
     "name": "测试项目",
     "description": "通过 API 创建的项目",
     "genre": "科幻"
   }
   ```
4. 点击"Execute"
5. 查看响应，记录返回的项目 ID

**测试 SSE 流式接口**：

1. 找到 `POST /api/ai/generate-stream` 接口
2. 这是一个 SSE 接口，返回流式数据
3. 可以使用 curl 测试：
   ```bash
   curl -N -H "Authorization: Bearer YOUR_TOKEN" \
     -H "Content-Type: application/json" \
     -d '{"prompt":"写一段关于修仙的描述","max_tokens":200}' \
     http://localhost:8000/api/ai/generate-stream
   ```

## 常见操作技巧

### 1. 快捷键

- `Ctrl + S`：保存当前章节
- `Ctrl + Z`：撤销
- `Ctrl + Y`：重做
- `Ctrl + F`：查找
- `Ctrl + H`：替换

### 2. 批量操作

- 按住 `Ctrl` 点击可多选角色/章节
- 按住 `Shift` 点击可范围选择
- 右键菜单提供批量操作选项

### 3. 自动保存

- 章节编辑器每 30 秒自动保存
- 可在设置中调整自动保存间隔
- 左下角显示保存状态
